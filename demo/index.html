Carbonware Flipbook

v1.0.0   A polished, cross-device PDF flipbook plugin. Vanilla JS, no framework, ~25 KB.

Auto-init usage

<link rel="stylesheet" href="cw-flipbook/src/carbonware-flipbook.css">
<script src="cw-flipbook/src/carbonware-flipbook.js" defer></script>

<button type="button"
        data-cwflip-pdf="/docs/brochure.pdf"
        data-cwflip-title="My Brochure"
        data-cwflip-pdfworker="/path/to/pdfjs/build/pdf.worker.mjs">
    Read the brochure
</button>

Programmatic usage

CWFlipbook.open({
    url: '/docs/brochure.pdf',
    title: 'My Brochure',
    workerSrc: '/path/to/pdfjs/build/pdf.worker.mjs',
    accent: '#0d9488',
    onPage: (n, total) => console.log(`Page ${n}/${total}`),
    onClose: () => console.log('closed')
});

Modes (auto-picked)

  • ≥ 1024 px: two-page spread with flip animation
  • 600-1023 px: two-page spread with slide transition
  • < 600 px: single page, swipe to navigate

Gestures

  • Swipe left/right → next/previous page
  • Pinch with two fingers → zoom (re-renders past 150% for crisp text)
  • Tap left/right edge → previous/next
  • Double tap → toggle fit ↔ zoomed

Keyboard shortcuts

  • ← / → — turn pages
  • + / − — zoom in/out
  • 0 — fit to page
  • F — fullscreen
  • Esc — close
